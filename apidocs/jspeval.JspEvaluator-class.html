Class JspEvaluator

source code

This class provides a way to evaluate JspModels. A JspSolution can be given to several functions to calculate the following metrics:

  • makespan
  • total tardiness
  • total setuptimes
  • load balancing
  • work in process (WIP)
  • flowtime

build_machine_assignment(self, solution)

source code 

This function assigns operations to machines according to a solution and calculates their priorities. A solution can be provided by an optimization-algorithm.

Parameters:
  • solution - a solution for this model, that determines the machines and the priorities
Returns: dict
a dictionary for every operation containing its determined machine and priority. Like: (job, op): (machine, prio)

execute_schedule(self, assignment)

source code 

This function takes an assignment, which must fit the model (i.e. it must have priorities for the correct number of operations) and calculates schedule for it. This means the time when a particular job is done is calculated from the assignment.

Parameters:
  • assignment (dict) - an dictionary holding the assigned machine and the calculated priority for every operation (in global notation). Like: (job, op): (machine, prio)
Returns: dict
a dictionary with every operation and its corresponding finishtime and the used setuptime as a tuple like: (job, op): (setup, finish)
